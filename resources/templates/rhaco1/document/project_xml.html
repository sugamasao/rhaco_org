アプリケーションXMLの構成 (setup/project.xml)

		
			
				管理者名
				メールの差出人などに使います

<project>

アプリケーションの本体

属性 説明
version アプリケーションのバージョン
name アプリケーションの名前
pathinfo エントリポイントのphpファイル名 (index.php) を定義すると.htaccessを生成できるようになる(アプリケーションの初期処理)

要素

<session> <cookie> <database> <define> <input> <password> <text> <select> <description>

<session>

セッションの定義

属性 説明
expire 有効期限(秒)
cache キャッシュリミッタを設定する(参考) [ none, nocache, private, private_no_expire, public ]

<cookie>

クッキーの定義

属性 説明
expire 有効期限(秒)

<define>

定数

属性 説明
name 定義名
value 値

<input>

設定値 Rhaco::constant(設定名) で参照する
setupで入力フィールドが表示される

属性 説明
type 設定名

要素

<title> <description> <data>

<password>

設定値 Rhaco::constant(設定名) で参照する
setupでパスワードフィールドが表示される

属性 説明
type 設定名

要素

<title> <description> <data>

<textarea>

設定値 Rhaco::constant(設定名) で参照する
setupでテキストフィールドが表示される

属性 説明
type 設定名

要素

<title> <description> <data>

<select>

設定値 Rhaco::constant(設定名) で参照する
setupでセレクトボックスが表示される

属性 説明
type 設定名

要素

<title> <description> <data>

<title>

タイトル

<description>

説明

<data>

データ(デフォルト値)

属性 説明
caption キャプション (<select>でのみ使用)

<"database">

データベースの設定

属性 説明
name 接続名

要素

<table> <ext> <map> <description>

<"table">

モデルの定義

属性 説明
name 接続名

要素

<table> <ext> <map> <default> <description>
