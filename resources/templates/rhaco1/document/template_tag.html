Templateの拡張タグ

<rt:template>

<rt:template>で囲まれた部分がテンプレートとして使用されます。 ひとつも<rt:template>がない場合は全体をテンプレートとして使用します。

属性 説明
name テンプレート名

<rt:extends>

継承するテンプレートを指定する

属性 説明
href 継承するテンプレートの相対パス

<rt:block>

上書き可能ブロックの定義、上書きを行う。 同名のブロックが<rt:block>で囲まれた部分に上書きされる

属性 説明
name ブロック名

<rt:include>

指定のテンプレートを展開する

属性 説明
href 展開するファイルパス

<rt:replace>

文字列を置換する

属性 説明
src 対象の文字列
dest 置換後の文字列

<rt:comment>

<rt:comment>で囲まれた部分が実行時に削除されるコメントブロックとなる

属性 説明
name ブロック名

<rt:loop>

<rt:loop>で囲まれた部分を繰り返し処理する。

属性 説明
param 対象の変数名
var paramからとりだされた現在の値を格納する変数名
key paramからとりだされた現在のキーを格納する変数名
counter offsetからの現在のカウント数を格納する変数名
offset 開始値
limit 処理する最大数

<rt:for>

<rt:for>で囲まれた部分を繰り返し処理する。

属性 説明
param 対象の変数名
var paramからとりだされた現在の値を格納する変数名
counter offsetからの現在のカウント数を格納する変数名
end 開始値
step 増数

<rt:else>

親要素の条件が偽の場合に親要素の<rt:else />の`下のブロック`を処理する

<rt:if>

条件が真の場合に<rt:if>で囲まれたブロックを処理する。 valueが指定されてない場合はparam値で判断する。

属性 説明
param 対象の変数名
value paramと比較する値

要素

<rt:else>

<rt:ifnot>

条件が偽の場合に<rt:ifnot>で囲まれたブロックを処理する。 valueが指定されてない場合はparam値で判断する。

属性 説明
param 対象の変数名
value paramと比較する値

要素

<rt:else>

<rt:hasnot>

対象の変数名が空( array(), null, '' ) の場合に囲まれたブロックを処理する。

属性 説明
param 対象の変数名

<rt:has>

対象の変数名が空( array(), null, '' ) では無い場合に囲まれたブロックを処理する。

属性 説明
param 対象の変数名

<rt:pager>

Paginatorを処理する。 <rt:pager>で囲まれたブロックがある場合はブロック内を処理する

属性 説明
param 対象のPaginator
navi 表示する要素を指定する (prev, next, first, last, counter)
counter カウンタの最大表示数
tag 各要素を囲むタグ名

<rt:invalid>

Exceptionsを処理する。 <rt:invalid>で囲まれたブロックがある場合はブロック内を処理する

属性 説明
name 対象のグループ名
var 対象のException配列を格納する変数名
type 各要素を囲むか ( ul, plan )

<form>

formの拡張

属性 説明
rt:reference formの要素にrt:referenceを適用する
rt:aref 結果（変数,タグの展開済）を処理する (true, false)

要素

<input> <textarea> <select>

<input>

<textarea>

Templateにセットされた値とname属性(またはid属性)を元にformを処理する。

属性 説明
rt:reference 変数を参照し処理する (true, false)

<select>

Templateにセットされた値とname属性(またはid属性)を元にformを処理する。 rt:paramから<option>を生成される。 rt:paramが未指定の場合はselect内要素が利用される。

属性 説明
rt:reference 変数を参照し処理する (true, false)
rt:param 対象の変数名
rt:null 空の値をもつ<option>を作成する、指定した文字列がラベルとなる

<table>

<ul>

<ol>

繰り返し処理の拡張を処理する

属性 説明
rt:reference 変数を参照し処理する (true, false)
rt:param 対象の変数名
rt:var paramからとりだされた現在の値を格納する変数名
rt:key paramからとりだされた現在のキーを格納する変数名
rt:counter offsetからの現在のカウント数を格納する変数名
rt:offset 開始値
rt:limit 処理する最大数
rt:null 空の値をもつ<option>を作成する、指定した文字列がラベルとなる

<iframe>

iframeの拡張

属性 説明
rt:reference formの要素にrt:referenceを適用する
