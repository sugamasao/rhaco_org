スタートアップ

ライブラリのインストールやアプリケーションのセットアップをrhaco3.phpで行うことができます。
主要な機能の他にライブラリが追加されると機能が増える場合があります。 

	php rhaco3.php

定義済みのコマンド

オプションを指定せずにrhaco3.phpを実行すると一覧が表示されます。

オプション 説明
import ライブラリをimport(インストール)する
phar ライブラリをひとつのphar(libs.phar)にまとめる
search ライブラリの検索
settings __settings__.phpを作成する
htaccess .htaccessを作成する

その他のライブラリ

例えばorg.rhaco.Manをインストールしてみます
			php rhaco3.php -import org.rhaco.Man

一覧にorg.rhaco.Manのセットアップが追加されています
			> php rhaco3.php
			
			Commands: 
			  import        : Download Package
			  phar          : Create a phar libs
			  search        : Search package
			  htaccess      : Create .htaccess
			  org.rhaco.Man : Class descriptions

オプションの接尾に?をつける事でセットアップの説明を見る事ができます
			> php rhaco3.php -org.rhaco.Man!!!?!!!

			Usage:
			  -org.rhaco.Man [(string) package path]
			
			  Options:
			    -m    (string) method name
			
			
			  description:
			    Class descriptions

org.rhaco.Objectの説明を見てみます
			> php rhaco3.php -org.rhaco.Man org.rhaco.Object

			class org.rhaco.Object:
			 Description:   基底クラス
			  Static methods defined here:
			    anon              : クラスのアノテーションを取得する
			    set_module        : クラスにモジュールを追加する
			
			  Methods defined here:
			    set_object_module : インスタンスにモジュールを追加する
			    prop_anon         : プロパティのアノテーションを取得する
			    props             : アクセス可能なプロパティを取得する

org.rhaco.Object::anonの説明を見てみます
			> php rhaco3.php -org.rhaco.Man org.rhaco.Object -m anon

			class org.rhaco.Object in method anon:
			 Description:
			   クラスのアノテーションを取得する
			
			 Parameter:
			    n  : [string]  アノテーション名
			    df : [mixed=null]  デフォルト値
			
			 Return:
			   mixed
