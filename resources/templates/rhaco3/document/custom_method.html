Objectのカスタマイズ

Object(org.rhaco.Object)を継承したクラスに特殊メソッドを定義することでObjectの挙動をカスタマイズする事ができます。

特殊メソッド ( protected )

メソッド名 説明
__new__ コンストラクタの拡張
__init__ インスタンス作成の拡張
__del__ デストラクタの拡張
__str__ オブジェクトの文字列表現の拡張

特殊メソッド ( static public )

メソッド名 説明
__import__ import時に実行される初期処理
__shutdown__ アプリケーションの終了時に実行される終了処理
