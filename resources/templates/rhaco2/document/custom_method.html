Objectのカスタマイズ

Objectを継承したクラスに特殊メソッドを定義することでObjectの挙動をカスタマイズする事ができます。

特殊メソッド ( protected )

メソッド名 説明
__hash__ $obj->hash()の拡張
__cp__ $obj->cp()の拡張
__new__ コンストラクタの拡張
__init__ インスタンス作成の拡張
__del__ デストラクタの拡張
__str__ オブジェクトの文字列表現の拡張

特殊メソッド ( static public )

メソッド名 説明
__import__ import時に実行される初期処理
__shutdown__ アプリケーションの終了時に実行される終了処理

ex..

		/**
		 * @var integer $id
		 */
		class Abc extends Object{
			static private $count = 0;
			protected $id;

			protected function __init__(){
				self::$count++;
				$this->id = self::$count;
			}
			protected function __str__(){
				return $this->id();
			}
			protected function __add__($arg){
				$this->id += ((int)$arg * 10);
			}
		}
		
		$obj = new Abc();
		println($obj);
		$obj = new Abc();
		println($obj);
		$obj = new Abc();
		println($obj);
		println($obj->add(4));

結果

		1
		2
		3
		43
