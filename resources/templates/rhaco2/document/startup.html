スタートアップ

アプリケーションの開発、利用はrhaco2.phpからはじまります。
主要な機能の他にライブラリが追加されると機能が増える場合があります。 

	php rhaco2.php

コマンドの説明を表示します

オプション 内容
-help import -importの説明を表示します

標準コマンド

オプションを指定せずにrhaco2.phpを実行すると一覧が表示されます。

オプション 説明
clean vendorsを削除しライブラリをimportする
config アプリケーションの設定を変更する
def 定義名のリスト
gettext po,moファイルを生成する
htaccess .htaccessを作成する
import ライブラリをimport(インストール)する
installation アプリケーションのインストレーションを表示する
log ログを制御する
man ソースドキュメントを表示する
new アプリケーションのひな形を作成する
test testを実行する

アプリケーションセットアップ

次回からsetup.phpを実行するとセットアップ画面が表示されアプリケーションの設定やテスト、単純なDBの操作等を行えます。
