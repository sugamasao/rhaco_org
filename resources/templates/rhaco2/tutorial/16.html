簡易HTTPサーバを利用する

org.rhaco.net.Runserver を利用しHTTPサーバを起動する

アプリケーションのひな形の作成

		php rhaco2.php !!!-new index!!!

 

org.rhaco.net.Runserver をインポートする

		php rhaco2.php -import org.rhaco.net.Runserver

 

Runserver をポート8080で起動する

		> php rhaco2.php -runserver -port 8080
		
		Development server is running at !!!http://localhost:8080!!!
		Quit the server with !!!CONTROL-C.!!!

 

		Runserver は control + c で終了します。
